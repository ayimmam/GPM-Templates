# Project Initiation Templates

This folder contains templates used during the **project initiation phase**, where project goals, scope, and stakeholders are formally defined.

These templates are foundational for:
- Project discovery
- Stakeholder alignment
- Business justification
- Early documentation in project management certificates

## When to Use These Templates
Use these documents at the start of a project, before detailed planning begins.  
They are commonly required artifacts in **entry-level project manager portfolios**.
