# Project Closure Templates

This folder includes templates used to **formally close a project and document outcomes**.

These artifacts are among the most valuable in a **project management portfolio**.

---

## Templates Included

### Project Closeout Report
Summarizes:
- Project objectives and outcomes
- Methodology used (Agile / hybrid)
- Planned vs actual scope, cost, and schedule
- Performance metrics
- Lessons learned and recommendations
- Ownership handoff

Sample reference:
- Sauce & Spoon Digital Tabletop Menu Project

Link:
- [Google Docs](https://docs.google.com/document/d/1otLB3HeAmVAQJlITUV3m4qgT_zwsyNwpqWIYBnHSgQ8/template/preview)

---

### Post Implementation Review
Used to:
- Capture insights from successes and failures
- Improve future projects
- Support organizational learning

Link:
- [Google Docs](https://docs.google.com/document/d/1YU_tsJKOuf0k7cqnPLkQ2ZvriT4b4B-Ota5Q-vuK2V0/template/preview)

## When to Use These Templates
Use these documents once deliverables are complete and the project is ready for handoff or evaluation.
